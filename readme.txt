Integration of banorte bank with woocommerce

== Description ==

Banorte Woocommerce is a payment method plugin for Woocommerce, integrates the banorte bank of Mexico allowing to receive payments through credit cards, Visa Mastercad

== Installation ==


1. Download the plugin
2. Enter the administrator of your WordPress.
3. Enter Plugins / Add-New / Upload-Plugin.
4. Find the plugin downloaded on your computer and upload it like any other file.
5. After installing the .zip you can see it in the list of installed plugins, you can activate or deactivate it.
6. To configure the plugin you must go to: WooCommerce / Adjustments / Finish Purchase and Locate the tab Banorte Woocommerce.
7. Make sure to fill in all the required fields of your configuration in relation to the access provided in your Banorte account


== Frequently Asked Questions ==

= Countries in which its use is available? =

= How do I prove its operation? =
